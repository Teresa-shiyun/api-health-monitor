\# API Health Monitor



A lightweight API monitoring dashboard built with Node.js, Express, SQLite and Chart.js.



\## Overview



This project periodically checks external APIs, records latency and status codes,

calculates uptime percentage, and visualizes performance in a real-time dashboard.



Inspired by reliability engineering and production monitoring systems.



\## Features



\- Scheduled health checks (node-cron)

\- Configurable endpoints (endpoints.json)

\- SQLite persistence

\- Uptime \& failure tracking

\- Latency visualization

\- Real-time dashboard updates



\## Tech Stack



\- Node.js

\- Express

\- SQLite

\- Chart.js

\- node-cron
